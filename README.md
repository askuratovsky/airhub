# Airhub app

Decoupled ruby microservice for [airport][airport] project.

## Run

Run airport app:
```shell
$ git clone git@github.com:askuratovsky/airport.git
$ cd airport
$ cp .env.sample .env
$ make run
```

Run airhub app:
```shell
$ git clone git@github.com:askuratovsky/airhub.git
$ cd airhub
$ make run
```

Open <http://localhost:3000>
